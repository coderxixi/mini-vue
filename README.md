## mini-vue

实现最简 vue3 模型，用于深入学习 vue3， 让你更轻松的理解 vue3 的核心逻辑
基于 vue3 的功能点，一点一点的拆分出来。
代码命名会保持和源码中的一致，方便大家通过命名去源码中查找逻辑。
